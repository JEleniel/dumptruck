# requirement:dual-mode-cli-server

## Name

Support CLI and HTTPS Server modes

## Type

requirement

## Status

approved

## Description

Dumptruck must support two primary runtime modes: a standalone CLI for ad-hoc analysis and an HTTPS server for remote analysis and job-oriented processing, with access controls suitable for operational deployment.
